A multi-model ML web app built with Streamlit that provides instant predictions across four real-world use cases from a single interactive interface.
🎯 Prediction Modules

🏠 House Price — Estimates property price based on area and bedrooms
🧍 BMI Calculator — Calculates BMI and classifies health category
💰 Insurance Cost — Predicts medical insurance charges based on age, BMI, and lifestyle
🚢 Titanic Survival — Predicts passenger survival probability by class, gender, and age

🛠️ Tech Stack
Python · Streamlit · Scikit-learn · NumPy · Pickle
